DATA FLOW:
----------
1. User interacts with React frontend
2. Frontend makes API call via Axios
3. Express backend processes request
4. Backend queries MongoDB or calls external API
5. Response sent back to frontend
6. Frontend updates UI

AUTHENTICATION FLOW:
--------------------
1. User submits credentials
2. Backend validates and hashes password
3. JWT token generated
4. Token stored in localStorage
5. Token included in API requests
6. Middleware verifies token

IMAGE ANALYSIS FLOW:
--------------------
1. User uploads image
2. Image converted to base64
3. Sent to Google Vision API
4. AI analysis results processed
5. Category and occasion matched
6. Item saved to MongoDB
