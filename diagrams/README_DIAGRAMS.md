## Quick Conversion Guide

### Using Draw.io (Recommended)
1. Go to https://app.diagrams.net/
2. Open existing diagram (`.drawio` files)
3. Or create new diagram using text/Mermaid as reference
4. Customize styling and layout
5. Export as PNG (for reports) or SVG (for vector graphics)

### Using Mermaid
1. Go to https://mermaid.live/
2. Paste Mermaid code from `.md` files
3. Preview the diagram
4. Export as PNG or SVG
5. Insert into report

### Using Other Tools
- **Lucidchart**: Import draw.io files or recreate manually
- **Microsoft Visio**: Recreate using text references
- **PlantUML**: Convert sequence diagrams if needed

## Tips

1. **Consistency**: Use similar colors and styling across all diagrams
2. **Clarity**: Ensure text is readable and diagrams are not cluttered
3. **Size**: Export at high resolution (at least 300 DPI for printing)
4. **Format**: PNG for bitmaps, SVG for vector graphics
5. **Labels**: Include all necessary labels and captions
6. **Colors**: Use colors that print well in grayscale (many reports are printed in B&W)
